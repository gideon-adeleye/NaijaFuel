# NaijaFuel 🍛
### Nigerian Calorie Tracker — OMAD Edition

A clean, premium calorie tracking web app built specifically for Nigerian meals and an OMAD (One Meal A Day) lifestyle. No backend, no accounts — just open and track.

## Features

- **30+ Nigerian foods** pre-loaded with accurate calorie and macro data (Jollof Rice, Egusi, Pounded Yam, Suya, Moi Moi, and more)
- **Quick search** — type any food and select from the database
- **Manual entry** — log any custom meal with full macro breakdown (calories, protein, carbs, fat, fiber)
- **One-click chips** — add common foods instantly
- **Visual calorie ring** — animated progress ring that turns red when you exceed your goal
- **Macro tracking** — protein, carbs, fat totals updated in real time
- **Custom calorie goal** — defaults to 1,500 kcal, fully adjustable
- **7-day streak tracker** — see which days you logged
- **Daily history** — last 30 days stored locally
- **Zero dependencies** — single HTML file, works offline
- **Persistent storage** — everything saved in localStorage, survives page refresh

## Usage

1. Clone or download the repo
2. Open `index.html` in any modern browser
3. Start logging

```bash
git clone https://github.com/yourusername/naijafuel.git
cd naijafuel
open index.html
```

No npm install. No build step. No server. Just open it.

## Why This Exists

Most calorie trackers don't understand Nigerian food. MyFitnessPal entries for "Egusi Soup" are wildly inconsistent. This app is built around real Nigerian meals with values grounded in nutritional research — designed for people eating the way Nigerians actually eat.
